Privacy Policy

Last Updated: March 28, 2026

Listing Genius AI ("the Extension") is committed to protecting your privacy. This policy explains what data the Extension accesses and how it is handled.

1. Data We Collect

We do not collect any personal data. The Extension does not have its own server and does not transmit any data to us.

2. Data Stored Locally

The following data is stored locally on your device using Chrome's storage API:

  • OpenAI API Key - Your API key, entered by you, stored locally to make API calls.
  • Generation History - Your past product listing generations (up to 50 entries).
  • Settings - Your preferred platform, language, tone, and AI model.
  • Usage Counter - Daily generation count for rate limiting.

This data never leaves your device except as described in Section 3.

3. Third-Party Services

The Extension communicates with the following third-party services:

  • OpenAI API (api.openai.com) - Your product description text is sent to OpenAI's API to generate listings. This communication is direct from your browser to OpenAI using your own API key. OpenAI's privacy policy applies to this data: openai.com/privacy
  • ExtensionPay (extensionpay.com) - Used for optional Pro subscription payment processing. ExtensionPay's privacy policy applies: extensionpay.com/privacy

4. Permissions Explained

  • storage - To save your settings, API key, and generation history locally.
  • activeTab - To detect which e-commerce platform you are on and auto-fill listing forms.
  • contextMenus - To provide right-click "Generate listing" functionality.
  • host_permissions (api.openai.com) - To make API calls to OpenAI for listing generation.

5. Data Deletion

You can delete all stored data at any time through the Extension's Settings page using the "Reset All Data" button. Uninstalling the Extension also removes all locally stored data.

6. Children's Privacy

The Extension is not directed at children under 13. We do not knowingly collect data from children.

7. Changes to This Policy

We may update this policy from time to time. Changes will be reflected in the "Last Updated" date above.

8. Contact

If you have questions about this privacy policy, please contact us at: support@listinggenius.ai
